# Sales Performance Analysis

## Problem Statement
Why is profit declining despite strong sales performance?

---

## Tools Used
- Excel (Data Cleaning & EDA)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)

---

## Process
1. Performed data cleaning and exploratory analysis in Excel  
2. Wrote SQL queries to analyze region, category, and discount impact  
3. Built interactive dashboards in Power BI  

---

## Key Insights
- West region is the highest performing region  
- Central region shows weak profitability  
- Furniture category is underperforming  
- Tables is the biggest loss-making sub-category  
- Profit declines sharply beyond 30% discount  

---

## Root Cause
High discounting on low-margin products (especially Tables) leads to losses.

## Recommendations

- Improve pricing strategy in Central region
